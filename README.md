# KoF_bot

Bot officiel du Royaume de France pour BitCraft Online.

## To Run (at project root):
Linux:

- `chmod +x ./run.sh` _(optional, if script isn't auth to execute)_
- `./run.sh`

Windows:

- Deprecated

## Required:
- Docker
- npm
  - npx

## Optional (for messing around with the DB outside the bot)
- SQLite3
